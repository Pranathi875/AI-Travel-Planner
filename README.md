#AI_Travel_Planner
This project is an AI-based travel planner that helps users plan trips by suggesting destinations, 
itineraries, and travel details based on user preferences.
#features
- Suggests travel destinations
- Generates itineraries
- User-friendly interface
- Personalized recommendations
#TechStack
-HTML
-CSS
-Python
#installation
1. Clone the repository:
   git clone https://github.com/Pranathi875/AI-Travel-Planner.git
2. Navigate to the project folder:
   cd AI-Travel-Planner
3. Install dependencies:
   pip install -r requirements.txt
#how to run
Run the project using:
python travel_planner.py
